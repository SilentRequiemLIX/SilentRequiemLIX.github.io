荣耀手机利用ADB全局锁120教学

2026-03-09

打开CMD，键入如下命令：

adb shell service call package 数字 s16 com.hihonor.android.hnaps i32 0 i32 0

上方命令中数字的获取：
安装MT管理器
定位到/system/framework
找到framework.jar文件
打开jar，选中所有dex文件，用dex编辑器++打开
点击搜索，发起新搜索，搜索以下内容

transaction_setsystemappinstall

点击唯一一个结果，找到结果旁边的十六进制值
将十六进制值转换成十进制就是上方命令所需要的数字了
如果要恢复，只需要执行

adb shell service call package 数字 s16 com.hihonor.android.hnaps i32 1 i32 0

以上只是第一个命令

adb shell settings put system peak_refresh_rate 你手机最大的刷新率

这一条命令是将你手机帧率设置为你想要的刷新率

adb shell settings put system min_refresh_rate 你手机最大的刷新率

这一条命令是将你手机最低刷新率改成你想要的刷新率
